Matrix screensaver

Pro linuxový X Window System máme program xscreensaver, který umožňuje nastavit matrixový screensaver. Prvně nainstalujeme xscreensaver a extra balíčky, kde je schovaný matrix styl.

sudo apt install xscreensaver xscreensaver-data-extra

Nastavit screensaver můžeme buď přes rozhraní xscreensaver-demo, nebo přímo v souboru ~/.xscreensaver. Např. pro náš matrix xmatrix -root -small -no-trace -delay 50000 -density 6.

Aby šel screensaver zapnout, tak musí program běžet (třeba na pozadí). Přidáme do ~/.profile následující inicializaci.

xscreensaver -no-splash &

Nakonec chceme nastavit klávesovou zkratku pro zamknutí obrazovky, tak nabindujeme následující příkaz.

xscreensaver-command -lock
