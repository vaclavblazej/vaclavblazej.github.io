Forged Alliance na Linuxu

Supreme Commander Forged Alliance je strategická PC hra. Tento návod Vám pomůže hru spustit pod lobby FAF, pokud máte hru koupenou na platformě Steam.

Jak na spuštění SC-FA

Program playonlinux umožňuje běh windowsovských programů pod linuxem (s pomocí wine). Výhodou je, že tento program spravuje verze wine a verze programů, je také možné vytvořit více různých virtuálních disků s různými konfiguracemi.

Pro spuštění Forged Alliance je potřeba playonlinux nainstalovat, v něm vytvořit disk, na něj nainstalovat FAF, Steam a z něho i SC-FA. FAF – Forged Alliance Forever, je herní lobby, které nahradilo oficiální servery (THQ skončila), a kde lze v dneštní době najít komunitu FA.

Kroky na spuštění SC-FA (2020)

  1. nainstalovat playonlinux sudo apt install playonlinux

  2. stáhnout dfc_windows-x64_1_1_1.exe na FAForever stránce ↗

  3. spustit playonlinux

  4. nainstalovat Wine v Tools/Manage Wine Versions a přidat nějaký (třeba 3.20)

  5. vybrat Install

    1. vespod vybrat – Install a non-listed program
    2. 3x Next
    3. Install a program in a new virtual drive, Next
    4. Vyplnit FA, Next
    5. Use another version of Wine a Install some libraries, Next
    6. Vybrat novou verzi wine (3.20 funguje), Next
    7. 64 bits windows installation, Next
    8. Zaškrtnout d3dx9, directx9, dotnet40, Next (čekat na nainstalování)
    9. Browse, a najít dfc_windows-x64_1_1_1.exe, Next, Next, Finish

  6. Z menu zástupců vybrat downlords-faf-client, Next, Next, a I don't want ..., Next

  7. Install, Gaming, Steam

  8. Spustit Steam, Přihlásit se, nainstalovat Supreme Commander: Forged Alliance

  9. Zapněte FAForever 4. je možné že dostanete error: You are missing the Microsoft Visual C++ 2010 Runtime. Stáhněte ze stránky: https://www.microsoft.com/en-us/download/details.aspx?id=5555 ↗ , tento soubor je pak potřeba stáhnout a nainstalovat na virtuální disk 4. Stačí tedy stáhnout ze zmíněné adresy (případně zde), a přes Configure, Miscellanious, Run a .exe in this virtual drive, vybrat soubor, nainstalovat

  10. Pokud na FAF nemáte účet, tak ho vytvořte a potvrďte emailem, pak se přihlašte 4. Jestli nenaskočila stránka FAF a místo ní vidíte jen holý program s čedým menu vespod … tak asi nejste připojeni k internetu, nebo nastala chyba, se kterou tento návod nepočítá

  11. (Nahoře) Options, Settings, Game path, a vybrat cestu (na virtuálním disku) ke složce hry

  12. V menu Play si vybrat, poklikat a hrát, vše by mělo už (více méně) fungovat

Nastavení hry pro fajnšmekry

  1. Nastavení zkratek je pod F1, jedno menu vyplnit pomocí nákresu.
  2. Rozkliknout levé menu pod surovinami, zaškrtat aby se dosahy všech věcí zobrazovali pořád.
