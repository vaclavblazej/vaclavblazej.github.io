Vánoční křížovka 2024

Během těchto vánoc jsem dal dohromady křížovku. Pěkné svátky, zde je (včetně zdroje), pro všechny k volnému užití.

K tomu přikládám zdrojový kód v LaTeX; křížovka je sázena pomocí TikZ. Dáno veřejně pod licencí public domain, tj volné užití pro cokoli bez nutnosti citace, avšak bez jakékoli záruky.

\documentclass[a4paper]{article}

\usepackage[utf8]{inputenc}
\usepackage{lmodern}
\usepackage[T1]{fontenc}
\usepackage{moresize}
\usepackage{anyfontsize}
\usepackage{setspace}

\usepackage[margin=2.3cm]{geometry}

\pagenumbering{gobble}

\newcommand\mainfontsize{\linespread{1.0}\ttfamily\fontsize{10pt}{10pt}\selectfont}
\newcommand\smallfontsize{\linespread{1.0}\ttfamily\fontsize{6pt}{6pt}\selectfont}

\renewcommand{\baselinestretch}{1.0}

% position
\newcommand{\mythickbox}[2]{%
    \node[draw,fill=white,line width=1.3pt,align=center,minimum width=10mm,minimum height=10mm] at (#1,-#2) {};%
}

\newcommand{\mytext}[3]{%
    \node[align=center,text width=10mm] at (#1,-#2) {%
        \smallfontsize%
        #3%
        \\%
    };
}

\newcommand{\mytextbox}[3]{%
    \mythickbox{#1}{#2}
    \mytext{#1}{#2}{#3}
}

\newcommand{\mytextboxy}[4]{%
    \def\val{%
        \smallfontsize%
        #3%
        \vspace{1pt}%
        \hrule%
        \vspace{1pt}%
        #4%
    }
    \mythickbox{#1}{#2}
    \mytext{#1}{#2}{\val}
}
