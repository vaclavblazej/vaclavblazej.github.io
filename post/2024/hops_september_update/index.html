Hierarchy of Parameters update 2

Another semester, another deployment of HOPS ↗. In the last update I mentioned a few things that ought to be done. The biggest update is addition of a good number of “unknown” sources and more precise bounds for relations in the main hierarchy. This step increased the number of source papers from 22 to 41. Some bounding relations are still missing (mainly connected to distance to cograph) and many incomparabilities are missing. This is mainly for relations connected to less popular parameters, which are now hidden in the main hierarchy. Also, a basic documentation about the concepts behind the parameter relations was added.

  • add a rule where from $A<B$ and $A<C$ we can infer $A<B+C$
  • fill in the blanks in the pairwise comparison table (vast majority is known, just unfilled right now)
  • export the data into json and make it available through a static API – as json files placed on particular addresses
  • fill in some handcrafted content for the more popular parameters
