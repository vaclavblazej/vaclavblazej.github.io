Editors

List of software that I find useful for various tasks.

  • text & code – I’ve used Vim for a while but now I’m using NeoVim; alternative would be vs code ↗ or JetBrains ↗ suite.
  • recording – Audacity ↗ with PulseEffects ↗ using my mic setup
  • document creation – Latex ↗
  • organization – Obsidian ↗, better than nothing; I used Roam Reserach for a while, it is worse.
  • video editing – Kdenlive ↗

I do not use the following often, or at all, but have some idea what I would try.

  • sound editing – rare, but would use Audacity ↗
  • music – if I wanted to get serious, then probably Cubase ↗; alternatively FL Studio ↗ or Reason ↗ (depends on usecase)
  • music dj – Traktor ↗, VirtualDJ ↗
  • sheet music – lilypond ↗
  • video automation – FFmpeg ↗
  • video editing – DaVinci Resolve ↗
  • 3d modelling – I really like Blender ↗
