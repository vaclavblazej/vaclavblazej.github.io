Youtrack setup

Youtrack ↗ je aplikace od JetBrains pro správu softwarových projektů. Je zdarma pro max 10 lidí, pokud si ji dotyční někde nasadí.

Stáhnutí a zapnutí

Stáhnout lze ze stránek programu ↗. Nejjednodušší je stáhnout .jar, které potom lze jednoduše spustit:

java -jar youtrack-7.0.27965.jar 8082

Poslední argument je port. Po spuštění lze najít stránky najít na adrese:

localhost:8082

Data se ukládají ve složce ~/teamsysdata/youtrack a jejich promazáním se vše resetuje jako před instalací. Také jsou nějaké pomocné informace ve složce ~/.youtrack, ale se zdají být pouze cache apod.

Nastavení

Youtrack vás uvítá obrazovkou Welcome, root! Nyní projdeme tedy tyto kroky,

  • inicializujeme: akceptovat licenci a nastavit heslo roota
  • vytvořme projekt: Projects, Create project, např. TEST a dát Scrum
  • nastavme time tracking: Projects, TEST, Edit project, Time tracking, Enable time tracking, Create and attach “Estimation” field, Create and attach “Spent time” field
  • nastavme board podle času: Agile Boards, Board Setting (tlačítko vpravo), Card, Estimation Field nastavit na Estimation

Tak a teď máme připravený systém pro trackování issue a práce.

Trýčky

  • Výběr Issue a Ctrl+Alt+J vyjede příkazovou řádku, pak je možné zadat př. práci takto: !work 1h20m" nebo můžeme i nastavit typ práce a popisek “work Development 1h20m my custom comment”.

Specs

youtrack-7.0
system: Ubuntu 16.10
